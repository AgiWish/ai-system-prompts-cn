Element Plus 本地环境冒烟测试

该页面使用本地 `Vue 3` 与 `Element Plus` 资源，集中测试基础组件、弹层组件、日期类组件和表格组件在本地直引模式下的表现。

建议优先观察：按钮是否渲染、下拉是否能展开、日期选择器是否能弹出、对话框是否可打开、日历和表格是否正常显示。

基础组件

用于验证样式、颜色、按钮、标签等最基础的渲染是否正常。
