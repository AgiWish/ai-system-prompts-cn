# 模块分桶（参考）

当作勾选清单使用——并非每个产品都需要每一行。不适用的整节可删；主题名称请改成贴合业务领域。

## A. 营销 / 内容官网

- 品牌与版式：设计系统、响应式断点、深色模式（若有）
- 核心页面：首页、产品/方案、定价、关于、联系、招聘等
- 内容：CMS 模型、博客/文档、媒体库、跳转与重定向、SEO（meta、sitemap、canonical）
- 获客与增长：表单、订阅通讯、试用/预约、推荐与 UTM 采集
- 信任与法务：隐私政策、用户协议、Cookie/同意横幅、地区免责声明
- 分析与运维：统计脚本、事件命名规范、实验/A-B 埋点（可选）

## B. Web 应用（登录后产品）

- 身份：注册/登录、SSO/SAML/OIDC、多因素、找回密码、会话与设备管理
- 角色与权限：RBAC/ABAC、组织/工作区、邀请加入
- 核心领域流程：产品存在价值所在（按用户任务拆成多条主题）
- 协作（若有）：评论、分享、@提醒、在线状态等
- 触达：邮件/站内信/Web Push、模板、通知偏好
- 媒体与文件：上传、杀毒/校验、配额、预览
- 搜索与发现：全局搜索、筛选、保存视图
- 集成：第三方 OAuth、Webhook、API Key、导入导出
- 计费（若有）：套餐、结账、发票、税务、欠费催收、用量计量
- 支持：帮助中心入口、联系客服、状态页链接

## C. 管理端 / 内部 / 运维

- 用户支持：代客操作（须写清政策！）、账号查询、审计轨迹
- 配置：功能开关、限流、公告
- 报表：导出、定时报表、个人信息处理规范
- 内容审核（若有 UGC）：队列、申诉、自动化规则

## D. 移动 App（原生 / 混合 / 响应式）

与 B 共用的能力可对齐映射；另加移动端特有项：

- 引导与权限：相机、麦克风、定位、通知、蓝牙等
- 离线与会话：冲突策略、排队、后台刷新
- 分发：签名、分阶段发布、最低系统版本
- 平台政策：App Store / Google Play 合规、账号删除、恢复购买等

## E. 横切非功能

- 安全：威胁模型摘要、密钥管理、Web 的 CSRF/CORS、移动端可选证书锁定
- 隐私：数据清单（高层）、保留策略、同意流程
- 无障碍：键盘、读屏、对比度、减少动效
- 可靠性：用白话写 SLA 或降级模式
- 本地化：RTL、日期与数字格式、文案冻结流程
