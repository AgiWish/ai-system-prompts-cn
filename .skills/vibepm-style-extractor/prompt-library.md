# 中文风格提示词模板库

以下模板用于“基于已提取风格DNA”生成新网页。
使用方法：把 `{}` 占位符替换为你的项目信息。

---

## 1) 极简专业（Minimal Pro）

### 基础提示词 / Foundation Prompt
请生成一个 {页面类型}，风格为“极简专业”。
保持克制、清晰、高可读性，避免装饰性噪音。
沿用以下风格约束：
- 颜色：以中性表面色为主，品牌色仅用于关键交互与状态强调
- 字体：标题与正文字体层级明确，字重对比明显但不过度
- 背景：纯色或非常轻微的线性渐变，不使用重纹理
- 阴影：低强度、短半径、少层级，仅用于区分悬浮层
- 边框：1px 细边框，低对比分割
- 圆角：统一中小圆角体系（如 8/12）
- 按钮：主/次/幽灵三种变体，状态反馈清楚
- 布局：12 栅格，8pt 间距节奏，大留白、低密度信息编排
避免：过度渐变、发光描边、玻璃拟态、大面积高饱和色块。

### 组件提示词 / Component Prompt
请生成一组组件：按钮、输入框、选择器、卡片、表格、弹层。
要求所有组件继承“极简专业”风格DNA，强调一致的边框、圆角、间距和字号层级；
交互态仅通过颜色、边框、阴影的小幅变化表达，不使用夸张动画。

### 变体提示词 / Variation Prompt
在不改变布局体系与组件几何规则的前提下，生成“更温和、更明亮”的同风格版本：
提升背景明度，降低对比冲突，品牌色饱和度下降 10%-15%，保持专业感和可读性优先。

## 2) 科技效率（Tech Focus）

### 基础提示词 / Foundation Prompt
请生成一个 {页面类型}，风格为“科技效率”。
视觉语言应理性、紧凑、信息优先，适合数据密集场景。
沿用以下风格约束：
- 颜色：深浅双层表面，冷色品牌强调，语义色明确
- 字体：偏现代无衬线，数字与标签可读性优先
- 背景：低干扰背景，允许轻微网格或微弱渐变
- 阴影：层级清晰但克制，避免霓虹泛光
- 边框：中低对比边框，数据区域分区明确
- 圆角：偏小圆角，强化“精确”感
- 按钮：强调操作效率，尺寸体系清楚，禁用花哨形态
- 布局：高信息密度模块化栅格，支持快速扫描
避免：赛博霓虹、无意义发光、装饰性图表、过大留白导致信息稀释。

### 组件提示词 / Component Prompt
请输出一组“科技效率”组件：筛选栏、数据卡、表格、分页、操作按钮、状态标签。
要求交互反馈明确，键盘焦点可见，状态语义（成功/警告/错误）对比充足，保证高效操作。

### 变体提示词 / Variation Prompt
保持信息密度与组件结构不变，生成“偏亮主题”的科技效率版本：
提升背景亮度、减弱阴影、增强分割线可见性，保留冷静理性的产品气质。

## 3) 轻奢高级（Soft Luxury）

### 基础提示词 / Foundation Prompt
请生成一个 {页面类型}，风格为“轻奢高级”。
气质应精致、克制、有质感，不过分炫技。
沿用以下风格约束：
- 颜色：低饱和中性色 + 单一高级强调色（如金属感暖色或深宝石色）
- 字体：标题可用高辨识度字形，正文保持高可读性
- 背景：细腻层次（轻渐变或极弱纹理）
- 阴影：柔和扩散阴影，营造温润层次
- 边框：细线与局部高光结合，强调精致边界
- 圆角：中等圆角，避免过于锐利或过分圆润
- 按钮：触感明确，强调“按下/悬停”的材质变化
- 布局：留白较多，模块节奏舒展，突出内容主次
避免：廉价金色、高饱和撞色、过度拟物和复杂特效。

### 组件提示词 / Component Prompt
请生成轻奢风组件：主按钮、次按钮、价格卡、信息卡、表单区块、导航。
要求统一材质语言（边框/阴影/高光逻辑一致），视觉重点收敛，细节精致。

### 变体提示词 / Variation Prompt
在不改变组件结构和层级体系的前提下，生成“更现代冷调”的轻奢版本：
将强调色向冷调偏移，降低暖色占比，保留高级和精致的整体气质。

## 4) 暗黑沉浸（Dark Immersive）

### 基础提示词 / Foundation Prompt
请生成一个 {页面类型}，风格为“暗黑沉浸”。
整体应沉稳、聚焦内容，强调夜间可读性与视觉舒适度。
沿用以下风格约束：
- 颜色：深色分层表面 + 受控高亮强调色
- 字体：中高对比字重层级，正文亮度不过曝
- 背景：深色纯底或微弱渐变，避免复杂纹理干扰
- 阴影：以分层对比为主，阴影不应脏灰或糊成一片
- 边框：深色边界中引入可感知分隔
- 圆角：中等圆角，保证现代感与秩序感
- 按钮：强调可点击性，focus ring 清晰
- 布局：模块分区明确，避免大面积同亮度导致层次丢失
避免：荧光霓虹堆叠、纯黑纯白极端对比、玻璃拟态泛滥。

### 组件提示词 / Component Prompt
请生成暗黑沉浸组件：按钮、输入、卡片、标签、侧边导航、顶部栏。
要求暗色下文本对比达标、状态色可区分、悬浮层层级清楚，且不依赖夸张光效。

### 变体提示词 / Variation Prompt
保持暗黑沉浸DNA不变，生成“更柔和夜间版”：
降低亮色对比峰值，增加中间灰阶层次，提高长时间浏览舒适度。

## 5) 编辑杂志感（Editorial）

### 基础提示词 / Foundation Prompt
请生成一个 {页面类型}，风格为“编辑杂志感”。
强调排版节奏、内容叙事和视觉呼吸感。
沿用以下风格约束：
- 颜色：以中性色和低饱和强调色构建内容优先的视觉系统
- 字体：标题与正文形成明显个性对比，建立强阅读层级
- 背景：干净背景为主，可有轻微纸感或内容区块底色
- 阴影：尽量弱化，以排版关系代替重阴影
- 边框：分割线用于组织信息，不喧宾夺主
- 圆角：按内容块属性使用小到中圆角，保持克制
- 按钮：样式简洁，强调文案与状态可辨识
- 布局：非完全对称的内容编排，允许“主次错位”的版式张力
避免：卡片瀑布流模板感、统一大圆角、图标堆砌和过度动效。

### 组件提示词 / Component Prompt
请生成编辑杂志感组件：文章卡、目录导航、标签筛选、引用块、订阅区、按钮。
要求优先表达内容层级与阅读节奏，减少装饰性UI噪声。

### 变体提示词 / Variation Prompt
在保持排版体系和内容层级不变的情况下，生成“更商业化”的版本：
提升行动按钮视觉权重，增强关键转化区对比，同时保留编辑感气质。

## 通用占位符建议

- `{页面类型}`：官网首页 / 产品页 / SaaS 仪表盘 / 博客首页 / 电商列表页
- 可追加：`{目标用户}`、`{品牌关键词}`、`{主要任务}`、`{终端优先级}`

## 组合技巧 / Composition Tips

1. 先用 `Foundation Prompt` 定调。
2. 再用 `Component Prompt` 统一组件语言。
3. 最后用 `Variation Prompt` 生成 A/B 风格分支。
4. 如果你已有提取结果，把 token 直接拼接在提示词末尾作为硬约束。
