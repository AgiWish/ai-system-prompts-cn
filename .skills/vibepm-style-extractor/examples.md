# 示例 / Examples

## 输入示例（URL）/ Example Input (URL)

用户请求：
"请分析这个站点风格，并给我一套可复用提示词：`https://example.com`"

期望行为：
1. 读取页面结构与风格信号。
2. 提取颜色、字体、阴影、边框、圆角、按钮、布局等 tokens。
3. 按固定格式输出，并包含提示词包。

## 输入示例（截图）/ Example Input (Screenshot)

用户请求：
"根据这张网页截图提取风格，并给我可直接用于生成页面的提示词。"

期望行为：
1. 从截图中推断近似 token 数值。
2. 对不确定数值使用 `~` 标注。
3. 提供 Foundation/Component/Variation 提示词，并包含反模式（anti-patterns）约束。

## 提示词包骨架示例 / Example Prompt Pack Skeleton

```markdown
### 基础提示词 / Foundation Prompt
创建一个现代金融科技仪表盘，整体风格克制且具有高级感。
使用深石板色表面、冷青色强调、适中对比的字体体系，
采用 12 栏响应式栅格、8pt 间距节奏、中等偏柔和圆角（10-14px），
使用分层但克制的阴影，并建立高可读、数据优先的信息层级。
避免霓虹发光、过度玻璃拟态以及过大的首屏 Hero 区块。

### 组件提示词 / Component Prompt
在同一设计 DNA 下生成按钮、卡片、输入框、表格和弹窗组件。
按钮：3 种尺寸，主/次/幽灵变体，具备清晰 hover/focus 状态。
输入框：明显的 focus ring、紧凑间距、中性边框与强调态焦点。
卡片：低对比边框 + 单一层级抬升阴影。

### 变体提示词 / Variation Prompt
保持布局与字体系统不变，但将配色整体向暖色偏移 10-15%，
并轻微提升表面对比度，以适配更明亮的日间使用场景。
圆角尺度与间距节奏保持不变。
```
