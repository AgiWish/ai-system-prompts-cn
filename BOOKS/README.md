# 经典书籍库

本目录用于把经典书籍、创业手册和长篇英文资料录入到同一个中英对照阅读器。

## 当前种子内容

- `founders-playbook/`：来自根目录 `创始人手册-构建一个AI原生初创公司.pdf` 的章节录入。
- 当前已按 PDF 目录拆分为 8 个 Markdown 文件：7 个正文章节 + 1 个资源页。
- 中文侧为章节级初译整理，状态标记为“初译待校对”，适合先用于中英对照阅读和后续人工校对。

## 完整录入前置条件

- 先确认原 PDF 的版权、授权和使用范围。
- 再决定是否允许在仓库中保存完整英文正文与中文译文。
- 若只能做个人学习用途，建议把完整拆分和译文保留在本地私有目录，不进入公开仓库。
- 若对外发布，应先完成版权确认和逐章人工校对，再同步调整对外说明。

## 录入约定

- 英文原文放在 `en/`。
- 中文译文放在 `zh/`。
- 同一章节的英文和中文文件名保持一致。
- 第一阶段以段落级对齐为主，不追求逐句级对齐。
- 版权状态未确认的资料只录入短样例或个人学习范围内容，不直接发布完整正文。
