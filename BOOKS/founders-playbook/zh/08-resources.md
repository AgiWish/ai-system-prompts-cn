## 使用 Claude 构建

• Building AI Agents for Startups：分享创业公司如何使用 Agent，在规模化过程中减少对创始人的依赖。

• Claude Code docs：从初始安装到高级 Agentic 工作流，为构建者提供路径。建议从 “How Claude Code works” 概览开始。

• Claude Code best practices：覆盖 Anthropic 内部和工程团队中有效的模式，包括上下文管理、权限、规划和验证工作流。

• Using CLAUDE.md files：讲解如何为具体代码库配置 Claude Code。对 MVP 阶段搭建开发环境的创始人尤其重要。

• Claude Code power user tips：来自 Claude Code 团队的工作流模式，包括并行会话和验证循环。

• Get started with Claude Cowork：介绍团队如何设置 Claude Cowork，并开始使用 skills、plugins 等能力，让它在创业公司中扩大影响。

• Tutorials：claude.com/resources/tutorials 提供可搜索的实操教程列表。

## 创始人故事

• How three YC startups built their companies with Claude Code：介绍 HumanLayer、Ambral 和 Vulcan Technologies 如何使用 Claude 快速推出原型，并扩展 AI 驱动平台。

• GC AI：创始人用领域知识构建了 Claude 驱动的法律平台，服务内部法务真实工作方式，包括公司特定 playbook、跨职能利益相关方，以及可变风险容忍阈值。

• Carta Healthcare：使用 Claude 驱动临床抽象平台，每年处理 22,000 个外科病例，并将数据抽象时间减少 66%。

• Anything：由 Claude 和 Agent SDK 驱动，帮助 150 万用户无需写代码就把想法变成可运行软件产品，其中包括一位非技术创始人构建并销售完整招聘平台。

• Cogent：一家应用 AI 实验室，构建 Agent 自动化关键企业安全任务。它使用 Claude 作为推理层，自动化漏洞生命周期中的调查、优先级排序和修复。

• Airtree：使用 Claude Cowork 作为运营基础设施中心，把过去散落在十几个工具和团队中的数据统一起来。现在，一个人用 skills 构建的工作流自动化，可以被整个组织复用。

• Duvo：构建运行采购、供应链和品类管理流程的 AI Agent，横跨 ERP、供应商门户、电子表格、邮件甚至电话。Duvo 完全基于 Claude 构建，并使用 Agent SDK 编排跨流程工作。

• Zingage：面向居家护理机构的 24/7 自动化运营 AI Agent 平台。它使用 Claude 的结构化工具调用跨 EMR 和多个沟通渠道编排，并用上下文推理给出更细腻、以患者为中心的结果。

• Kindora：由一位非营利组织高管使用 Claude Sonnet 构建的 AI 平台，用于智能匹配慈善组织和资助方。筛掉成千上万低价值匹配后，Kindora 的 MCP connector 让非营利组织可以直接在 Claude 中使用它的 prospecting 工具。

• Wordsmith：由律师转型 CTO 创办，为内部法务团队提供可靠 AI 法律技术。Claude 是 Wordsmith 合同审查、协议起草和文档审阅能力的推理引擎，工程团队也使用 Claude Code 构建和演进平台。
